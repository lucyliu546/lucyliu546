About Technologies

Inspiration:

As a casual baker sometimes a recipe will call for something I do not have on hand. This leads to some googling to see if a substitute ingredient exists. This was my inspiration for creating this web app to cut down the amount of time searching around and spend more time on baking!

About:

Use the search box to search for an ingredient that you need a substitute for and the quantity. Additionally there is a pantry feature to see if you have the substitue ingredients readily available.

Backend:

Python, Django, Django Rest Framework, Postgres

Frontend:

React, HTML, CSS, Material UI
